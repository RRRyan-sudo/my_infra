# 关于DSA

Data Structure & Algorithm，简称DSA。

本DSA仓库的定位是用于应聘，准备笔试或面试手撕。

内容如下，

1. LeetCode HOT100
2. 剑指offer
3. 各种面经整理出的题

--- 

跋

* 我走的社招，主要拷打的是工作内容和项目经历，所以认为DSA掌握以上内容就已经足够了。

* 学习过程：拿到题快速确定考察点，看答案，Python写一遍（标答），部分C++实现用AI生成后看一遍。

* 2刷及以后：看题——（2分钟内会写）Python独立手写——（2分钟内没思路）看标答对比——C++运行，检查bug。

* 谨防背题，初次学习量可以大些，有个大概的印象，手写部分可以扣细节，手写过程中回味下标答给出的解题思路。

* 差不多3遍下来，基础面试不至于没思路，可能会存在debug拖慢面试节奏的情况，不用慌，面试官也想水时长。

* 面试时，真遇到写不出来的，做好心理建设，对方可能就是不招，找个难题筛掉你，不必为写不出懊恼。

* 从业方向是AI大模型、多模态、具身智能，涉及模型类的手撕可以看Transformer仓库。
